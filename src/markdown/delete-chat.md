# Borrado del chat

Detecta cuando el streamer o un moderador ha borrado el chat.

| Parámetro | Descripción |
|-|-|
| `type` | Tipo de evento: `clearchat`. |
| `roomId` | ID del canal. |
| `tmi` | Timestamp del momento en el que ocurre el evento. |
| `raw` | Mensaje sin procesar. |
